Special Effects

This dialog is not available when the CUPS driver (Mac OS X v.10.6.x, Mac OS X v.10.5.x) is used.

This dialog allows you to set functions that apply various effects to image data such as photos and correct colors.

You can print a photo like a hand-drawn illustration or print it with a single color such as sepia, which allows you to create printouts somewhat different from ordinary photos. The printer also provides functions that print photos in more vivid color and reduce noise from images recorded with digital cameras.

Simulate Illustration

Using Simulate Illustration allows you to print image data like a hand-drawn illustration.

Checking the check box enables simulate illustration. If necessary, adjust contrast by moving the Contrast slider. To restore to the original image data, uncheck the check box.

Monochrome Effects

Using Monochrome Effects allows you to present image data with a single color.

Checking the check box enables monochrome effects. Select your desired color from Sepia, Blue, Pink, Green, or the color selected using Select Color. To restore to the original image data, uncheck the check box.

Vivid Photo

Using Vivid Photo allows you to print photo with vivid colors. The vivid photo function emphasizes the colors in background sceneries while maintaining the human skin color natural. By using this function, you can make vivid hues appear even more vivid.

Check this check box to enable the vivid photo function. To return to the original images colors, uncheck this check box.

Important

  • With Mac OS X v.10.5.x, the vivid photo function is disabled when ColorSync is selected for Color Matching, even when Vivid Photo is selected.

    With Mac OS X v.10.4.11, the vivid photo function is disabled when you select ColorSync from the Color Correction menu in the Color Options dialog, even when Vivid Photo is selected.

Note

  • When you are using the CUPS driver (Mac OS X v.10.6.x, Mac OS X v.10.5.x), you can select Color Options on the pop-up menu of the Print dialog, and then select Vivid Photo for Color Mode.

Photo Noise Reduction

This function reduces digital camera noise and improves the image quality of digital prints.

To enable this function, check this check box and select an effect, either Normal or Strong. To disable this function, uncheck this check box.
