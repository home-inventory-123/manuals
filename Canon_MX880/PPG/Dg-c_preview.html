Displaying the Print Results before Printing

You can display and check the print result before printing.

To display the print result before printing, click Preview on the Print Dialog.

Note

  • With Mac OS X v.10.6.x or Mac OS X v.10.5.x, when you click (the disclosure triangle) in the Print dialog and switch to the detailed display, the Preview button is not displayed.
    For detailed display, a preview is displayed on the left side of the Print dialog.
