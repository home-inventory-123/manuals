Simulating an Illustration

This function is not available when the CUPS driver (Mac OS X v.10.6.x, Mac OS X v.10.5.x) is used.

With the Simulate Illustration function, you can print full-color or 256-color image data so that it looks like a hand-drawn illustration. This function adds different effects to the original profile and colors.

The procedure for performing Simulate Illustration is as follows:

  1. Select Special Effects from the pop-up menu on the Print Dialog

  2. Set the Simulate Illustration

    Check the Simulate Illustration check box, and if necessary, adjust Contrast.
    Moving the slider to the right lightens the image data and moving the slider to the left darkens the image data.
    The selected contrast will be reflected in the preview on the left side of the printer driver.

  3. Complete the setup

    Click Print.
    When you execute print, the image will be printed so that it looks hand-drawn.
