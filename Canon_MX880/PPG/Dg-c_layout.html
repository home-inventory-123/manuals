Page Layout Printing

The page layout printing function allows you to print more than one page image on a single sheet of paper.

The procedure for performing page layout printing is as follows:

If you are using Mac OS X v.10.6.x or Mac OS X v.10.5.x

  1. Click (the disclosure triangle) in the Print Dialog

    The setup window switches to the detailed display.

  2. Select Layout from the pop-up menu

  3. Set the number of original you print on one page

    In Pages per Sheet, specify the number of page you print on one page.

  4. If necessary, set the following items

    Layout Direction

    To change the page placement order, select an icon from the list.

    Border

    To print a page border around each document page, select a type of page border.

    Reverse page orientation (Reverse Page Orientation)

    Select this check box to change the paper orientation.

    Flip horizontally

    Select this check box to reverse left and right of the document.

    Important

    • With the printer driver (Mac OS X v.10.5.x), you cannot use Flip horizontally.

  5. Complete the setup

    Click Print.
    When you execute print, the specified number of pages will be arranged on each sheet of paper in the specified order.

If you are using Mac OS X v.10.4.11

  1. Select Layout from the pop-up menu on the Print Dialog

  2. Set the number of original you print on one page

    In Pages per Sheet, specify the number of page you print on one page.

  3. If necessary, make the following settings

    Layout Direction

    To change the page placement order, select an icon from the list.

    Border

    To print a page border around each document page, select a type of page border.

  4. Complete the setup

    Click Print.
    When you execute print, the specified number of pages will be arranged on each sheet of paper in the specified order.

Important

  • When page layout printing is enabled, you cannot perform borderless printing.
