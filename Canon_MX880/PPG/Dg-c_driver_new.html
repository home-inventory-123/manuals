Obtaining the Latest Printer Driver

By updating the printer driver to the latest version of the printer driver, unresolved problems may be solved.

Checking the Printer Driver Version

After you open System Preferences and select Print & Fax, you can check the printer driver version according to the following procedure.

  • If you are using Mac OS X v.10.6.x or Mac OS X v.10.5.x

    When you select your model and click Options & Supplies..., the version of the printer driver is indicated in the Driver Version on the General tab.

  • If you are using Mac OS X v.10.4.11

    Select your model, and then click Printer Setup....
    When you select Name & Location from the pop-up menu, the version of the printer driver is indicated in the Driver Version.

Obtaining the Printer Driver

You can access our website and download the latest printer driver for your model.

Important

  • You can download the printer driver for free, but any Internet access charges incurred are your responsibility.
