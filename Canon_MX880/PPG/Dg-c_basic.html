Printing with Easy Setup

The simple setup procedure for carrying out appropriate printing on this machine, is as follows:

If you are using Mac OS X v.10.6.x or Mac OS X v.10.5.x

  1. Select the printer

    Select your model from the Printer list in the Print Dialog.

  2. To specify detailed settings, click (the disclosure triangle)

    The setup window switches to the detailed display.

  3. Select the paper size

    For Paper Size, select the paper size to be used.
    If necessary, set the number of copies, the pages to be printed, and the orientation.

  4. Select Quality & Media from the pop-up menu

  5. Select the media type

    For Media Type, select the same paper type loaded in the machine.

  6. Select the paper source

    Select Automatically Select, Rear Tray, Cassette, Continuous Autofeed, or Paper Allocation that matches your purpose for Paper Source.

    Important

    • The paper source settings that can be selected may differ depending on the paper type and size.

  7. Select the print quality

    For Print Quality, select High, Standard, or Fast, according to your purpose.

  8. Complete the setup

    Click Print.
    When you execute print, the document will be printed in accordance with the type and size of the media.

If you are using Mac OS X v.10.4.11

  1. Select the printer

    In the Page Setup Dialog, select your model name from the Format for list.

  2. Select the paper size

    Select the size of the paper to be used for printing from the Paper Size menu, and click OK.

  3. Select Quality & Media from the pop-up menu of the Print Dialog

  4. Select the media type

    For Media Type, select the same paper type loaded in the machine.

  5. Select the paper source

    Select Automatically Select, Rear Tray, Cassette, Continuous Autofeed, or Paper Allocation that matches your purpose for Paper Source.

    Important

    • The paper source settings that can be selected may differ depending on the paper type and size.

  6. Select the print quality

    For Print Quality, select High, Standard, or Fast, according to your purpose.

  7. Complete the setup

    Click Print.
    When you execute print, the document will be printed in accordance with the type and size of the media.
