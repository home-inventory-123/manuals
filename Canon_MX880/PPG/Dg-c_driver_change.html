Drivers Installed on Mac OS X v.10.5.x

The following two printer drivers are installed on Mac OS X v.10.5.x:

  • CUPS driver that operates on Mac OS X v.10.5.x and later

  • Same printer driver installed on Mac OS X v.10.4.11

In this topic, the drivers above are indicated as "CUPS driver" and "printer driver" respectively.

With default settings, the CUPS driver is used for printing.
Using the CUPS driver enables you to print in the High Dynamic Range (16-bit work flow for each color) and to print from 64-bit application software.

Restrictions on the CUPS Driver

The following restrictions apply when the CUPS driver is used for printing. Please note these points before you use the CUPS driver.

  • The Special Effects option in the Print dialog cannot be used.

  • With Mac OS X v.10.5.3 and later, if you connect this product and AirPort with a USB cable and print, you must install the latest AirPort firmware.

  • Printing cannot be executed with a Bluetooth connection.

Switching Between the CUPS Driver and the Print Driver

The CUPS driver and the print driver can be installed in the same system.

To be able to switch drivers when printing, prepare a print queue for each driver according to the following procedure.

  1. Display the printer list

    Open System Preferences, and select Print & Fax.
    Click + to display the printer list.

  2. Select the driver to be used

    From the printer list, select "Canon your model name", and set the driver as described below.

    • To use the CUPS driver

      For Print Using, select "Canon your model name".
      To distinguish the driver from the application software, change the printer name displayed in Name to an easily recognized name such as "CUPS your model".

    • To use the printer driver

      For Print Using, select Canon IJ Printer. The item that has "IJ" in its name is the printer driver.

  3. Complete the setup

    Click Add.

    The setup procedure is complete. If necessary, you can now switch the printer driver in the application software, and begin printing.
