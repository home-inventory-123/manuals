Printing with ICC Profiles (Specifying an ICC Profile from the Printer Driver)

Print from an application software that cannot identity input ICC profiles or does allow you to specify one by using the color space of the input ICC profile (sRGB) found in the data.

If you are using Mac OS X v.10.6.x or Mac OS X v.10.5.x

  1. Select Quality & Media from the pop-up menu of the Print Dialog

  2. Select the media type

    For Media Type, select the same media type loaded in the machine.

  3. Select the print quality

    For Print Quality, select High, Standard, or Fast, according to your purpose.

  4. Select color correction

    Select Color Matching from the pop-up menu, and then select ColorSync.
    You can leave Profile set to Automatic.

    Important

    • When you select a Profile setting other than Automatic or a printing ICC profile paper, the printer will not be able to print with the correct colors.

  5. Complete the setup

    Click Print.
    When you execute print, the printer uses the color space of the image data.

Important

  • You cannot specify an input ICC profile from the printer driver.

If you are using Mac OS X v.10.4.11

  1. Select Quality & Media from the pop-up menu of the Print Dialog

  2. Select the media type

    For Media Type, select the same media type loaded in the machine.

  3. Select the print quality

    For Print Quality, select High, Standard, or Fast, according to your purpose.

  4. Select color correction

    Select Color Options from the pop-up menu, then select ColorSync from the Color Correction menu.

  5. Complete the setup

    Click Print.
    When you execute print, the printer uses the color space of the image data.

Important

  • You cannot specify an input ICC profile from the printer driver.
