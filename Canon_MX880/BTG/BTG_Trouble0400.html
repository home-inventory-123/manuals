Canon IJ Printer Utility Does Not Open

  • Check: Is the Canon IJ Printer Utility opened by the Printer Setup Utility?

    When you open the Canon IJ Printer Utility to perform an operation such as maintenance via Bluetooth communication, begin by opening the Printer Setup Utility.

    • In Mac OS X v.10.6.x:

      1. Make sure that the printer is turned on.

      2. Select System Preferences from the Apple menu.

      3. Click Print & Fax.

      4. Select the device name of the printer to which the Bluetooth unit is attached from Printers.

      5. Click Open Print Queue....

      6. Click Print Setup.

      7. Click Utility.

      8. Click Open Printer Utility.

        When the Canon IJ Printer Utility opens, perform necessary maintenance.

    • In Mac OS X v.10.5.x:

      1. Make sure that the printer is turned on.

      2. Select System Preferences from the Apple menu.

      3. Click Print & Fax.

      4. Select the device name of the printer to which the Bluetooth unit is attached from Printers.

      5. Click Open Print Queue....

      6. Click Utility.

        When the Canon IJ Printer Utility opens, perform necessary maintenance.

    • In Mac OS X v.10.4.11:

      1. Make sure that the printer is turned on.

      2. Select Applications from the Go menu of Finder.

      3. Double-click the Utilities folder and the Printer Setup Utility icon.

      4. Select the device name of the printer to which the Bluetooth unit is attached from Name, and click Utility.

        When the Canon IJ Printer Utility opens, perform necessary maintenance.
