Installing the MP Drivers

Note

  • If you use the printer connecting with a computer, the MP Drivers have already been installed. In this case, move to the next step.
    Checking the Device Name of the Printer

Before connecting the Bluetooth unit to the printer for wireless printing, first the USB cable must be connected between the printer and computer, and the MP Drivers must be installed.

If you reinstall the MP Drivers, perform Custom Install on the Setup CD-ROM, then select MP Drivers.
