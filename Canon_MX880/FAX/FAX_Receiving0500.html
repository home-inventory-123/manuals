F021

Memory Reception

If the machine receives a fax under one of the following conditions, the received fax will not be printed and will be stored it in machine's memory. The FAX Memory lamp is lit and Received in memory. is displayed on the LCD.

If the machine receives a fax under one of the following conditions, the received fax will be automatically stored in machine's memory.

  • The machine is set to store received faxes in its memory:

    When Do not print is selected for Received documents in Auto print settings under FAX settings, the received faxes are stored in machine's memory.

    You can print the faxes stored in machine's memory from the Memory reference screen in the Fax menu.

    Printing a Document in Machine's Memory

    Note

    • To print the received faxes automatically, select Print for Received documents in Auto print settings under FAX settings.

      Auto print settings

  • An error has occurred on the machine:

    If you select Print for Received documents in Auto print settings under FAX settings, a received fax will be printed automatically. However, if any of the following errors occurs, the received fax will not be printed. It will instead be stored in machine's memory.

    When the error is resolved, the fax stored in machine's memory is printed automatically.

    • The amount of remaining ink is insufficient to print a fax:

      Check the ink level and take the appropriate action.

      Routine Maintenance

      Note

      • You can set the machine to forcibly print a received fax even if the ink has run out.

        However, part or all of the fax may not be printed since the ink has run out.

        Also, the contents of the fax will not be stored in machine's memory.

        If the ink has already run out, we recommend that Do not print be selected for Received documents in Auto print settings under FAX settings to store the received fax in machine's memory. After you check the ink level, take the appropriate action, and select Print for Received documents in Auto print settings, the fax stored in machine's memory will be printed automatically.

        Auto print settings

    • Paper has run out:

      Load the paper and press the OK button.

    • A different size of paper from that specified by Page size is loaded:

      Load the same size of paper as that specified by Page size, then press the OK button.

    • Page size is set to other than A4, Letter, or Legal (MX420 series):

      Set Page size to A4, Letter, or Legal.

      Changing the Print Settings

    • Type is set to other than Plain paper (MX420 series):

      Set Type to Plain paper.

      Changing the Print Settings

  • You pressed the Stop button to cancel printing of a fax:

    Press the FAX button to resume printing of the fax.

Note

  • MX880 series can store up to approximately 250 pages* (max. 30 documents) of faxes in machine's memory.

    MX420 series can store up to approximately 50 pages* (max. 20 documents) of faxes in machine's memory.

    * When using ITU-T No.1 chart (Standard mode)

  • If the machine's memory becomes full, the machine cannot receive any more faxes. Print the received faxes stored in machine's memory or save them on the memory card/USB flash drive, delete them from the machine's memory, and have the sender resend the faxes.
