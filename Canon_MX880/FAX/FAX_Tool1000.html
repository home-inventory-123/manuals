Uninstalling Speed Dial Utility

Follow the procedure below to uninstall Speed Dial Utility.

  1. Select Applications from the Go menu of Finder, double-click Canon Utilities > Speed Dial Utility, then drag the Speed Dial Utility icon into the trash.

Note

  • To reinstall Speed Dial Utility, uninstall Speed Dial Utility, then perform Custom Install with the Setup CD-ROM and select Speed Dial Utility to install it again.
