N056

Cannot Access to the Memory Card in the Card Slot Using MP Navigator EX

Note

  • In Mac OS X v.10.6.x or Mac OS X v.10.5.x, you cannot access to the Card Slot using MP Navigator EX over a network.

If "Cannot access the memory card" is displayed when you click Memory Card under Scan/Import Documents or Images on MP Navigator EX, confirm what the message says, then click Retry. If the message appears again, mount the Card Slot as the network drive. For details on how to mount the Card Slot, see Mounting the Card Slot as the Network Drive. Confirm the icon for the Card Slot appears on the desktop, and try to access using MP Navigator EX.
