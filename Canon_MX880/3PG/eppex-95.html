How Can I Move (or Copy) the Saved File?

If you want to move (or copy) a file created and saved with Easy-PhotoPrint EX from one folder to another, you need to move (or copy) the folder that was automatically created when originally saving that file as well.

For example, when you save a file named "MyAlbum.el1," a folder named "MyAlbum.el1.Data" is automatically created in the same folder that contains the "MyAlbum.el1" file. If you want to move (or copy) the "MyAlbum.el1" file to another folder, move (or copy) the "MyAlbum.el1.Data" folder as well. The "MyAlbum.el1.Data" folder contains the photos used in the album.

Note

  • The icons may vary depending on the items.

Important

  • Do not change the Data folder name; otherwise you will not be able to display the photos you edited with Easy-PhotoPrint EX.

  • For PREMIUM Contents Print using premium content images, if you save the file and move it to another computer, you will not be able to open the file.
