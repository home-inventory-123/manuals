Adjusting Images

You can make fine adjustments to the overall brightness, contrast, etc. of images.

  1. Scan documents into MP Navigator EX and save them, then open the View & Use window from the Navigation Mode screen and select the photos you want to adjust.

    Note

    • See "Let's Try Scanning" to scan images into MP Navigator EX.

    • You can also select images saved on a memory card or computer.

      Importing Images Saved on a Memory Card

      Opening Images Saved on a Computer

  2. Click Edit/Convert, then click Fix photo images on the list.

    The Correct/Enhance Images window opens.

    Note

    • The Correct/Enhance Images window can also be opened by clicking (Image Correction/Enhancement) on the Toolbar or in the Zoom in dialog. In that case, only the target image (outlined in orange) can be corrected/enhanced.

    • See "Correct/Enhance Images Window" for details on the Correct/Enhance Images window.

  3. Select the image you want to adjust from the thumbnail list.

    The selected image appears in Preview.

    Note

    • If you selected only one image in the View & Use window, the thumbnail list does not appear and only the preview image appears.

  4. Select Manual, then click Adjust.

  5. Move the slider of the item you want to adjust and set the effect level.

    When you move a slider, (Correct/Enhance) mark appears on the upper left of the thumbnail and preview image.

    Note

    • Click Advanced to make fine adjustments to the image brightness and color tone. See "Advanced" in the descriptions of the Correct/Enhance Images window for details.

    • Click Defaults to reset all adjustments.

    • Click Reset Selected Image to cancel all corrections, enhancements, and adjustments applied to the selected image.

  6. Click Save Selected Image or Save All Corrected Images.

    You can save adjusted images as new files.

    Note

    • To save only the images you like, select them and click Save Selected Image. To save all images, click Save All Corrected Images.

    • The file format of adjusted images is JPEG/Exif.

  7. Click Exit.

    Important

    • The adjustments will be lost if you exit before saving adjusted images.
