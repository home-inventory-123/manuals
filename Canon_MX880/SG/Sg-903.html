ScanGear (Scanner Driver) Does Not Start

Check 1: Make sure that the ScanGear (scanner driver) is installed.

If it is not installed, insert the Setup CD-ROM into the computer's disc drive, then select Custom Install and install MP Drivers.

Check 2: On the File menu of the application, select Select Source and select the machine.

Note

  • The operation may differ depending on the application.

Check 3: Make sure that the application is TWAIN-compliant.

You cannot open ScanGear from applications not supporting TWAIN.

Check 4: Scan and save the image using MP Navigator EX, then import it into the application.

You can specify which application to start after scanning, in Application Settings of the Custom dialog of MP Navigator EX.
