Deleting the Unnecessary ScanGear (Scanner Driver)

You can delete the ScanGear (scanner driver) you no longer use.

When deleting ScanGear, close all running applications and remove the USB cable connecting the machine and computer.

  1. Delete ScanGear.

    From the Go menu of Finder, select Computer.

    On the hard disk on which the ScanGear is installed, browse to Library folder, Image Capture folder, TWAIN Data Sources folder, then drag and drop the "Canon XXX series.ds" (where "XXX" is the model name) and "Canon XXX series Network.ds" (where "XXX" is the model name) into the trash can.

  2. Restart the computer.

    Empty the trash can and restart the computer.

Important

  • ScanGear cannot be deleted if you are not logged in as a user with administrator privileges.

    For more about administrator privileges, refer to the Macintosh User's Guide.
