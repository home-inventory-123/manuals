E-mail Software Program You Want to Use Does Not Appear in the Screen for Selecting an E-mail Software Program

Check 1: MP Navigator EX may not support the e-mail software program.

The following e-mail software programs are supported. (Images are attached to the mail message automatically.)

- Mail (Mac OS standard)

- Microsoft Entourage

Check 2: If using an e-mail software program other than the ones listed above, select None (Attach Manually) when prompted to select a program and attach the scanned image manually.
