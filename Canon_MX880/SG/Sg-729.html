Print Photo Dialog

Click Print in the View & Use window, then click Print Photo on the list to open the Print Photo dialog.

Important

  • The Print Photo dialog will not open if Easy-PhotoPrint EX is installed. Easy-PhotoPrint EX starts instead.

  • PDF files cannot be printed.

Note

  • Double-click an image to display it in a different window.

  • Page Setup...

    Opens the page setup dialog of the operating system. You can specify the printer and paper size.

    For details, refer to the help file of the operating system.

  • Printer Properties...

    Opens the printer properties dialog of the operating system. You can make advanced print settings.

    For details, refer to the help file of the operating system.

  • Print

    Starts printing.

    Note

    • Only the images in the Thumbnail window with the checkbox selected are printed.

  • Close

    Closes the dialog without printing the photo.
