What Is MP Navigator EX (Supplied Scanner Software)?

MP Navigator EX is an application that enables you to easily scan photos and documents. It is suitable even for beginners.

Important

  • MP Navigator EX may not start from the Operation Panel of the machine. In that case, restart the computer.

  • MP Navigator EX can scan images of the following data size:

    - 10000 pixels x 10000 pixels or less

Note

  • ScanGear (scanner driver) may not operate properly in the Classic environment. In that case, stop the Classic environment.

What You Can Do with This Software

This software allows you to scan multiple documents at one time, or scan images larger than the Platen. You can also save scanned images, attach them to e-mail or print them.

Screens

There are two types of MP Navigator EX Main Menu: Navigation Mode screen and One-click Mode screen.

Navigation Mode Screen

You can start various tasks from the Navigation Mode screen, including simple scanning, scanning using ScanGear, and enhancing/correcting images.

One-click Mode Screen

You can complete from scanning to saving, etc. at one time by simply clicking the corresponding icon in the One-click Mode screen.

Scan/Import Window

Use the Scan/Import window to scan photos and documents, or import images saved on memory cards.

View & Use Window

Use the View & Use window to select what you want to do with the scanned images.

Note

  • See "File Formats" for data formats (extensions) supported by MP Navigator EX.
