Opening the Page Setup and Print Dialog

You can open the Page Setup dialog and the Print dialog from the application program you are using.

Opening the Page Setup Dialog

Open the Page Setup dialog to specify page (paper) settings before printing.

  1. Select Page Setup... on the File menu in your application program.

    The Page Setup dialog opens.

Opening the Print Dialog

Open the Print dialog to specify print settings before printing.

  1. Select Print... on the File menu in your application program.

    The Print dialog opens.
