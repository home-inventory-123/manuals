U307

An Unintended Language Is Displayed on the LCD

Select the language you want to be displayed according to the following procedure.

  1. Press the Setup button, then wait for about 5 seconds.

  2. Select Device settings, then press the OK button.

  3. Press the button 6 times, then press the OK button.

    If the Bluetooth unit is connected to the machine, press the button 8 times, and then press the OK button.

  4. Use the button to select the language for the LCD, then press the OK button.
