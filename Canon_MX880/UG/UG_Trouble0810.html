U347

Cannot Install the Application Program

Easy-WebPrint EX Does Not Start Up

If Easy-WebPrint EX does not start up on Safari, confirm the following.

Note

  • You can use Easy-WebPrint EX with the following environment.

    • OS: Mac OS X v.10.5.x or more

    • Browser: Safari v.4.x

  • Check: Is Easy-WebPrint EX installed on your computer?

    Make sure that Easy-WebPrint EX is installed on your computer following the procedure below.

    1. Select Applications from the Go menu of Finder.

    2. Double-click Canon Utilities then Easy-WebPrint EX.

    3. Make sure that the Easy-WebPrint EX.app file is in the Easy-WebPrint EX folder.

      If the Easy-WebPrint EX.app file is not in the folder, the Easy-WebPrint EX is not installed on your computer. Perform Custom Install on the Setup CD-ROM, then select Easy-WebPrint EX to install it onto your computer.

      Note

      • If Easy-WebPrint EX is not installed on your computer, the Easy-WebPrint EX download icon may appear in the Dock when the Solution Menu EX is running. Click the icon and follow the on-screen instructions to install.

      • While the installation or download Easy-WebPrint EX is in progress, it is necessary to access the Internet. A connection charge is taken upon yourself.
