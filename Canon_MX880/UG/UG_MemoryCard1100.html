Printing All Photos Saved on a Memory Card/USB Flash Drive (Print all photos)

You can print all photos saved on the memory card or USB flash drive.

  1. Display the card menu screen.

    How to Select a Card Menu

  2. Select Print all photos, then press the OK button.

  3. Load paper.

  4. Specify the print range.

    Use the button to select the method of specifying the print range, then press the OK button.

    The following methods are selectable.

    Select date

    The screen to select the shooting date is displayed. The photos in the selected date range are printed.

    Use the button to select the shooting date and the OK button to fix the start and end of the date. After fixing the end of the date, the print settings confirmation screen is displayed.

    Note

    • The date to be specified is displayed according to the settings of Date display format in Device user settings under Device settings.

      Device user settings

    All photos

    All photos saved on the memory card or USB flash drive are printed.

    After selecting the method and pressing the OK button, the print settings confirmation screen is displayed.

  5. Specify the number of copies by using the + or - button.

    Specify the print settings as necessary.

    Setting Items for Printing Photographs

  6. Press the Color button.

    The machine starts printing all the photos in the specified range.
