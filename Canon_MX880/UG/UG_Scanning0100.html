Before Scanning

Before scanning images, confirm the following:

Make sure that the original to be scanned meets the requirements for an original to be set on the Platen Glass or in the ADF.

If you forward the data to the computer, refer to Placing Documents for details.

If you save the data to the memory card or the USB flash drive, refer to Loading Paper / Originals for details.
